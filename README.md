# monorepo-js

[![Build Status](https://travis-ci.org/Himenon/monorepo-js.svg?branch=master)](https://travis-ci.org/Himenon/monorepo-js)

Practice of lerna

## Usage

### Initialize

```
yarn install
yarn bootstrap
```

### Build

```
yarn build
```

### Test

```
yarn test
```

### Development of each package

```
cd packages/bar-pkg
yarn install
yarn start
```
